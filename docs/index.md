# scgpt-neurodegen-probe

*A zero-shot probe of single-cell foundation models on neurodegeneration tasks where the ground truth is biologically known — built to be defensible even when the answer is negative.*

This documentation site mirrors the [README on GitHub](https://github.com/arminbayati/scgpt-neurodegen-probe#readme),
with additional pages for the biology primer, the architecture walkthrough,
the auto-generated API reference, and the full ADR-style decision log.

Start with **Biology primer** if you are an ML reader; start with **Architecture**
if you are a biologist or hiring manager scanning for what the code does.
